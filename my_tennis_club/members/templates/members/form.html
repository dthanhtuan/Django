How Django Forms Work:

  • {% csrf_token %} - Required for POST requests (Rails does this automatically)
  • {{ form.as_p }} - Renders entire form as paragraphs
  • {{ form.as_table }} - Renders form as table rows
  • {% for field in form %} - Manual control over each field
  • Form validation - Handled in forms.py (like Rails strong params)

Rails Comparison:

# Rails:
<%= form_for @member do |f| %>
  <%= f.text_field :firstname %>
  <%= f.submit %>
<% end %>

# Django:
<form method="post">
  {% csrf_token %}
  {{ form.as_p }}
  <button type="submit">Submit</button>
</form>
